## 배열 Int -> String 
로또 번호가 배열 속에 있는데 이 배열을 벗기기 위해서 
joined 사용하는법을 알게 되었다.
[Int] 를 스트링으로 변환 시키고  스트링 값을 연결하여 문자열을 만드니 인트처럼? 되었다. 
[Int] -> [String] ->  "" 없는 String 출력

## 딕셔너리 
키와 벨류 값으로 나타난다. 
var dir = [Int: String]()
dir = [1 : "a", 2: "b"] 
print(dir[1]) = 이런식으로 키값에 접근하면 벨류값을 알아 낼수있다.
