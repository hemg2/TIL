## week3 에 대해서
프로퍼티 메서드에 대해서 알고 있었지만. 먼가 조금더 객체속에 있는 값들이랑 친해진 느낌을 얻었습니다.
간단하게 정리를 하게 된다면
변수 => 값을 저장하는 공간
프로퍼티 => 객체가 들고있는 변수
인스턴스 => 객체를 담은 변수
즉 인스턴스도 변수, 프로퍼티도 변수
함수 => 동작
메서드 => 객체안에 있는 함수
즉 메서드도 함수
객체 == 틀
인스턴스 == 틀로만든 구현체
이정도로 정리를 할 수 있을 꺼같습니다.
